Ich habe eine erste funktionsfähige Version des Parsers fertiggestellt. 
Sie besteht aus einer Hauptklasse (Parser) und einer Klasse für die 
Fehler (Error), welche jedoch auch an anderer Stelle verwendet werden 
kann. Das ganze funktioniert dann so, dass man ein Parser-Objekt anlegt, 
welches den Code, den man dem Konstruktor mitgibt überprüft und 
eventuell auftretende Fehler in einem Vector speichert. Diese Fehler 
können dann abgerufen werden.
